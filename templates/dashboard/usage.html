How to use your API key

Gateway URL

Use {{ request.scheme }}://{{ request.get_host }}/ollama/ as your Ollama API base URL and include your API key in the headers.

Authentication

You can authenticate using any of these headers:

  • Authorization: Api-Key YOUR_API_KEY
  • Authorization: Bearer YOUR_API_KEY (compatible with OpenAI SDKs)
  • X-API-Key: YOUR_API_KEY

Replace YOUR_API_KEY with the key shown after creation (starts with oah_).

curl - Generate

  -H "Authorization: Api-Key YOUR_API_KEY" \
  -d '{

  "prompt": "Why is the sky blue?",
  "stream": false
}'
curl - Chat 
curl {{ request.scheme }}://{{ request.get_host }}/ollama/api/chat \
  -H "Authorization: Api-Key YOUR_API_KEY" \
  -d '{
  "model": "gemma3:1b",
  "messages": [{"role": "user", "content": "Hello!"}],
  "stream": false
}'
Python - OpenAI SDK 
from openai import OpenAI

client = OpenAI(
    base_url="{{ request.scheme }}://{{ request.get_host }}/ollama/v1",
    api_key="YOUR_API_KEY",
)

response = client.chat.completions.create(
    model="gemma3:1b",
    messages=[{"role": "user", "content": "Hello!"}],
)
print(response.choices[0].message.content)
Python - requests 
import requests

response = requests.post(
    "{{ request.scheme }}://{{ request.get_host }}/ollama/api/chat",
    headers={"Authorization": "Api-Key YOUR_API_KEY"},
    json={
        "model": "gemma3:1b",
        "messages": [{"role": "user", "content": "Hello!"}],
        "stream": False,
    },
)
print(response.json()["message"]["content"])
